# Grok Stock Analyzer

A beautiful, real-time stock dashboard for **TSLA, AMD, AAPL, NVDA**.

## Features
- Live prices and percentage changes
- Interactive price charts (1M / 3M / 6M / 1Y)
- Latest news for each ticker
- Grok-style AI opinion
- Global market indices snapshot
- Macro impact analysis

## How to use
1. Download or clone this repository
2. Open `index.html` in your browser (Chrome/Firefox recommended)
3. No installation or API keys needed

**Note**: This uses Yahoo Finance public endpoints (client-side only). Heavy usage may hit rate limits — click "Refresh All Data" sparingly.
